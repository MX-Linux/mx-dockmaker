MX Dockmaker Help

MX Dockmaker is a graphical tool for creating and editing application docks for MX Fluxbox.

Creating a Dock

  1. Add applications one at a time.
  2. Select a .desktop file or enter a command manually.
  3. Choose the icon size, background colors, and border colors.
  4. Select Add application for additional items, or Save to finish.

Editing Existing Docks

When the program starts, choose whether to create, edit, move, or delete an existing dock. Editing loads the saved dock so you can adjust commands, icons, tooltips, styling, and item order.

Moving and Reordering

You can move an existing dock to a different slit location from the startup mode dialog. Inside the main editor, drag dock icons with the mouse to reorder them.

Saving

Saving writes the dock file and attempts to add it to the Fluxbox menu. After saving, launch the dock to confirm the icons, commands, and styling behave as expected.

Notes

  • Desktop entries usually provide the correct application name and icon automatically.
  • For custom commands, choose a custom icon if the default executable icon is not suitable.
  • If you apply style changes to all items, the current appearance settings are propagated across the dock.
